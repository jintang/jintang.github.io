### setInterval(function,time)
此方法是在time的延迟之后才可以循环执行，而不是马上开始循环执行
### cacl()
这是css3中常用到的计算方法，但是当要使用`+ - * /`等运算符的时候，运算符两边要使用**空格**隔开，否则不起作用，如下是正确的方法：
``` css
.box{
    width:cacl(100% - 10px);/*运算的具体数值要带单位*/
}
```
